## 产品概述

将"幸运"模块的交互形式升级为幸运转盘（Spin Wheel）抽奖，提升视觉体验和交互乐趣。

## 核心功能

- 幸运转盘交互：将当前按钮点击抽奖改为旋转轮盘抽奖，支持动态分段（根据配置的任务数量自动调整轮盘扇区）
- 旋转动效：点击轮盘中心按钮，轮盘旋转并自然减速停止，最终指向随机选中任务
- 结果展示：旋转停止后，高亮显示选中任务，展示在轮盘下方的任务卡片中
- 今日任务持久化：保留每日抽取一次的逻辑，已抽取今日任务后不可再抽
- 配置入口：保留跳转到任务配置页的入口

## 页面布局（从上到下）

1. 页面标题区域（"幸运任务" + 副标题）
2. 幸运转盘区域（核心交互区，包含指针、转盘、中心旋转按钮）
3. 今日任务展示区域（已抽取时展示任务卡片，未抽取时展示提示信息）
4. 配置提示区域（跳转到任务配置页）

## 技术栈

- **Vue 3** + **TypeScript** + **SCSS**（复用项目现有技术栈）
- 数据持久化：`localStorage`（复用现有 Storage 工具）
- 动画：纯 CSS `transform: rotate()` + `cubic-bezier()` 缓动函数

## 实现方案

### 核心策略

使用 **SVG + CSS Transform** 实现幸运转盘，无需引入 Canvas 或第三方库，与项目现有技术栈完全兼容。

### 架构设计

```
lucky.vue（主页面）
  ├── LuckyWheel.vue（转盘组件）
  │     ├── SVG 绘制轮盘扇区（基于任务数量动态生成）
  │     ├── 固定指针指示器（SVG 三角形）
  │     ├── 中心旋转按钮（点击触发旋转）
  │     └── CSS transform: rotate() 动画
  ├── 今日任务展示（同现有逻辑）
  └── 配置提示入口
```

### 技术细节

#### 1. 转盘绘制（SVG 方案）

- 使用 SVG `<path>` 元素绘制扇区弧线
- 每个扇区角度 = 360 / 任务数量
- 扇区颜色从预设的鲜艳调色板中循环选取（8种渐变色）
- 每个扇区中心位置绘制任务描述文字（通过 `textPath` 或旋转 `text` 元素实现）

#### 2. 旋转动画

- 监听中心按钮 click 事件
- 随机计算目标旋转角度：当前角度 + 随机偏移（确保随机性）+ 多圈旋转（最少 4 圈，最多 8 圈）
- 使用 CSS `transition: transform 4s cubic-bezier(0.15, 0.85, 0.35, 1.05)` 实现自然减速效果
- 旋转结束后通过 `transitionend` 事件触发结果展示

**关键设计**：使用 `cubic-bezier(0.15, 0.85, 0.35, 1.05)` 缓动函数模拟物理惯性，先快后慢再轻微回弹，视觉上最接近真实转盘。

#### 3. 结果判定

- 维护一个累计角度 `currentRotation`
- 计算最终停止角度对应的扇区索引
- 与选中任务关联并展示

#### 4. 边界情况处理

- **任务数量为 0**：禁用旋转按钮，显示"还没有配置任何任务"提示
- **今日已抽取**：进入页面时检查 localStorage，如果已有今日任务，轮盘不可旋转，显示"今日已抽取"
- **任务数量频繁变动**：每次轮盘初始化时从 `Storage.luckyTasks.getAll()` 实时读取
- **单任务**：只有一个任务时，轮盘只有 1 个扇区，仍保持旋转动效增加趣味性
- **任务数量过多（> 12）**：任务较多时文字可能重叠，限制最大显示 12 个扇区（优先选择最近添加的），其余任务通过"还有 X 个神秘任务"提示

#### 5. 今日任务逻辑（保持不变）

- 每日首次进入时可抽取一次
- 抽取后存入 `localStorage` key `lucky_today_task`
- 次日 0 点重置可抽取状态

### 性能优化

- 转盘使用 SVG 而非 Canvas，无需额外的渲染上下文
- 旋转动画使用 CSS `transform`（合成器属性），不触发重排/重绘
- 使用 `will-change: transform` 提示浏览器优化
- 扇区文字过大时自动缩进省略（通过 SVG `textLength` 或 `textLength` + `lengthAdjust` 控制）

### 兼容性

- 项目已有深色模式支持，转盘颜色在深色模式下调整饱和度
- 移动端 touch 事件适配

### 转盘视觉效果

- 每个扇区使用纯净明亮的渐变色（紫、蓝、粉、橙、绿、靛、红、青 8 色调色板）
- 扇区间用细白线分割，增强视觉清晰度
- 扇区中心位置显示任务文字，白色加文字阴影确保可读性
- 旋转时扇区颜色微微发光，增强动态感
- 中心按钮为圆形紫色渐变，带有脉冲光晕动画

### 交互反馈

- 点击旋转按钮：按钮轻微下沉 + 脉冲放大效果
- 旋转过程中：指针微微闪烁
- 停止时：选中扇区发光放大 + 下方卡片滑入动画
- 不可旋转时：按钮变灰并显示锁定图标

### 深色模式适配

- 轮盘背景从白色变为深蓝紫色
- 扇区颜色饱和度略微降低，保持视觉舒适度

## 扩展使用计划

本项目无需使用任何扩展，所有功能通过现有的 Vue 3 + SCSS 技术栈即可实现，不涉及文件生成、浏览器操作或第三方服务集成。
