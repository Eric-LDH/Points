# 状态栏深色模式完全适配指南

## 📋 问题描述

**现象：**
- Vue3 移动端 APP 支持深色/浅色模式切换
- 页面内容（文字、背景、组件）都能正确变色
- **但手机屏幕最顶部（听筒/刘海/状态栏区域）的背景色始终是白色**
- 即使切换到深色模式，状态栏仍然是白色，非常突兀

**原因：**
- 状态栏区域（刘海/听筒位置）的背景色由**原生 Android 系统**控制
- Web 端的 CSS 只能控制 WebView 内部，无法直接控制原生状态栏
- 必须通过 Capacitor StatusBar 插件或原生代码来设置

## ✅ 完整解决方案

### 方案架构

```
┌─────────────────────────────────────┐
│      状态栏区域 (原生控制)           │
│  ┌───────────────────────────────┐  │
│  │   WebView 内容区 (Web 控制)     │  │
│  │                               │  │
│  │   内容延伸到状态栏下方         │  │
│  └───────────────────────────────┘  │
└─────────────────────────────────────┘
```

#### 2️⃣ 在 App.vue 中初始化状态栏

✅ **已完成** - 已在 `src/App.vue` 中添加：

```typescript
import { StatusBar, Style as StatusBarStyle } from '@capacitor/status-bar'

// 初始化状态栏样式
const initStatusBar = async () => {
  try {
    const capacitor = (window as any).Capacitor
    if (capacitor && capacitor.isNativePlatform()) {
      // 设置状态栏透明，让 WebView 内容延伸到状态栏下方
      await StatusBar.setStyle({ style: StatusBarStyle.Dark })
      await StatusBar.setBackgroundColor({ color: store.darkMode ? '#1F2937' : '#F9FAFB' })
      await StatusBar.setOverlaysWebView({ overlay: true })
      console.log('状态栏初始化成功')
    }
  } catch (error) {
    console.error('初始化状态栏失败:', error)
  }
}

// 监听深色模式切换，同步更新状态栏
watch(darkMode, async (newVal: boolean) => {
  await updateStatusBarStyle(newVal)
}, { immediate: false })

// 更新状态栏样式
const updateStatusBarStyle = async (isDark: boolean) => {
  try {
    const capacitor = (window as any).Capacitor
    if (capacitor && capacitor.isNativePlatform()) {
      // 更新状态栏背景色
      await StatusBar.setBackgroundColor({ 
        color: isDark ? '#1F2937' : '#F9FAFB' 
      })
      
      // 更新状态栏图标颜色
      await StatusBar.setStyle({ 
        style: isDark ? StatusBarStyle.Light : StatusBarStyle.Dark 
      })
      
      console.log('状态栏样式已更新:', isDark ? '深色' : '浅色')
    }
  } catch (error) {
    console.error('更新状态栏样式失败:', error)
  }
}
```

## 🎨 工作原理

### 深色模式切换流程

```
用户切换深色模式
    ↓
store.darkMode 变化
    ↓
触发 watch(darkMode) 监听
    ↓
调用 updateStatusBarStyle(isDark)
    ↓
通过 StatusBar 插件设置原生状态栏
    ├─ setBackgroundColor() - 背景色
    └─ setStyle() - 图标颜色
    ↓
状态栏完美适配深色/浅色模式 ✅
```

### 颜色配置

| 模式 | 背景色 | 图标颜色 |
|------|--------|----------|
| **浅色模式** | `#F9FAFB` (浅灰) | `StatusBarStyle.Dark` (深色图标) |
| **深色模式** | `#1F2937` (深灰) | `StatusBarStyle.Light` (浅色图标) |

## 🔍 关键配置说明

### 1. StatusBar.setStyle()

```typescript
StatusBarStyle.Dark   // 深色图标（适合浅色背景）
StatusBarStyle.Light  // 浅色图标（适合深色背景）
```

**注意：** Capacitor 8.x 使用 PascalCase（首字母大写）

### 2. StatusBar.setBackgroundColor()

```typescript
await StatusBar.setBackgroundColor({ 
  color: '#1F2937'  // RGB 十六进制，不带透明度
})
```

### 3. StatusBar.setOverlaysWebView()

```typescript
await StatusBar.setOverlaysWebView({ overlay: true })
```

- `true`: WebView 内容延伸到状态栏下方（推荐）
- `false`: WebView 在状态栏下方，不重叠

### 4. Android 原生配置关键点

```xml
<!-- 状态栏透明，才能用代码控制背景色 -->
<item name="android:statusBarColor">@android:color/transparent</item>

<!-- 允许绘制到系统栏区域 -->
<item name="android:windowDrawsSystemBarBackgrounds">true</item>

<!-- 不自动添加 padding，让内容延伸到顶部 -->
<item name="android:fitsSystemWindows">false</item>

<!-- 支持刘海屏 -->
<item name="android:windowLayoutInDisplayCutoutMode">always</item>
```

## ⚠️ 注意事项

### 1. 仅在原生环境生效

- StatusBar API **只在 Android/iOS 原生 APP 中有效**
- 在浏览器开发时不会报错，但不会有视觉效果
- 使用 `capacitor.isNativePlatform()` 判断环境

### 2. 权限配置

检查 `AndroidManifest.xml` 是否有必要权限（通常不需要额外权限）：

```xml
<manifest xmlns:android="http://schemas.android.com/apk/res/android">
  <!-- StatusBar 不需要额外权限 -->
</manifest>
```

### 3. 华为设备特殊处理

华为鸿蒙系统可能需要额外配置：

```xml
<!-- styles.xml -->
<item name="android:forceDarkAllowed">true</item>
```

如果仍然有问题，可以在 `MainActivity.java` 中添加：

```java
@Override
public void onCreate(Bundle savedInstanceState) {
    super.onCreate(savedInstanceState);
    
    // 强制启用深色模式
    if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.Q) {
        getWindow().setNavigationBarContrastEnforced(false);
    }
}
```

### 4. 颜色格式

- 使用 **6 位十六进制**（如 `#1F2937`）
- 不要带透明度通道（alpha）
- StatusBar 会自动使用完全不透明
