# 状态栏适配 - 快速部署步骤

## 🚀 立即执行（3 步完成）

### 第 1 步：同步代码到 Android

```bash
npm run cap:android
```

或者分步执行：

```bash
npm run build
npx cap sync android
```

### 第 2 步：打开 Android Studio

```bash
npx cap open android
```

### 第 3 步：运行 APP

在 Android Studio 中：
1. 点击顶部工具栏的 **Run** 按钮（绿色三角形 ▶️）
2. 或使用快捷键：**Shift + F10**
3. 选择你的测试设备（真机或模拟器）

### 如果仍然有问题

#### 问题 1：状态栏还是白色

**解决方法：**
```bash
# 清理并重新构建
cd android
.\gradlew clean
cd ..
npm run build
npx cap sync android
```

然后在 Android Studio 中重新运行。

#### 问题 2：控制台报错

查看浏览器控制台或 Android Logcat，检查是否有以下错误：

**错误：`StatusBar is not available`**
- 原因：在浏览器中运行，而不是原生 APP
- 解决：必须通过 `npx cap run android` 或 Android Studio 运行

**错误：`Cannot read property 'setBackgroundColor' of undefined`**
- 原因：插件未正确安装或同步
- 解决：重新执行 `npm run cap:android`

#### 问题 3：华为设备特殊处理

如果你的华为手机仍然显示白色状态栏：

**在 `MainActivity.java` 中添加：**

打开文件：
```bash
code android/app/src/main/java/com/points/family/MainActivity.java
```

添加代码：
```java
package com.points.family;

import android.os.Build;
import android.os.Bundle;
import com.getcapacitor.BridgeActivity;

public class MainActivity extends BridgeActivity {
    @Override
    public void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        
        // 华为设备强制启用深色模式
        if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.Q) {
            getWindow().setNavigationBarContrastEnforced(false);
        }
    }
}
```

然后重新运行。

## 💡 核心原理

```
┌──────────────────────────────┐
│   原生状态栏 (Android 控制)   │ ← StatusBar 插件控制
├──────────────────────────────┤
│   WebView (Web 内容)          │ ← CSS/Vue 控制
│                              │
│   内容延伸到状态栏下方        │
└──────────────────────────────┘
```

**关键点：**
1. Web 端无法直接控制原生状态栏
2. 必须使用 Capacitor StatusBar 插件
3. 需要原生配置配合（透明背景、延伸内容）
