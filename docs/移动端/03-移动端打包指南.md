## 🎯 方案一：Capacitor + Android Studio（强烈推荐）

### ✅ 优势
1. **零代码改动** - 现有项目完美支持
2. **现代化技术栈** - Ionic 团队维护
3. **开发体验佳** - 支持热重载调试
4. **性能优秀** - 使用系统 WebView
5. **国际通用** - 全球开发者广泛使用

### 📋 前置条件
1. ✅ Node.js v16+
2. ✅ Android Studio
3. ✅ Android SDK（API 30+）
4. ✅ JDK 11+

### 🚀 快速打包流程

#### 方法 A：使用 PowerShell 脚本（最简单）
```powershell
# 在项目根目录执行
.\build-android.ps1
```

脚本会自动完成：
- ✅ 检查环境
- ✅ 构建 Web 应用
- ✅ 同步到 Android
- ✅ 提示下一步操作

#### 方法 B：手动执行命令
```bash
# 1. 构建 Web 应用
npm run build

# 2. 同步到 Android
npm run cap:android

# 3. 打开 Android Studio
npm run cap:open:android

# 4. 在 Android Studio 中打包
#    File → Project Structure → 修改版本号
#    Build → Generate Signed Bundle / APK
#    选择签名密钥 → 生成 APK
```

#### 方法 C：命令行一键打包
```bash
# Debug 版本（测试用）
npm run apk:debug

# Release 版本（发布用）
npm run apk:release
```

### 📦 APK 输出位置
- **Debug**: `android\app\build\outputs\apk\debug\app-debug.apk`
- **Release**: `android\app\build\outputs\apk\release\app-release-unsigned.apk`

### 🔐 签名配置

#### 生成签名密钥
```bash
keytool -genkey -v -keystore my-release-key.keystore -alias alias_name -keyalg RSA -keysize 2048 -validity 10000
```

#### 配置签名（capacitor.config.ts）
```typescript
android: {
  buildOptions: {
    keystorePath: 'path/to/your.keystore',
    keystorePassword: 'your_password',
    keystoreAlias: 'alias_name',
    keystoreAliasPassword: 'your_password',
    releaseType: 'APK'
  }
}
```

### 💡 常见问题

#### Q1: Gradle 构建失败
```bash
# 解决方案：
1. 打开 Android Studio → Tools → SDK Manager
2. 安装 Android SDK Platform (API 30+)
3. 安装 SDK Build-Tools
4. 重启 Android Studio
```

#### Q2: 白屏问题
```bash
# 解决方案：
1. 确认已执行 npm run build
2. 检查 dist 目录是否有文件
3. 清理缓存重新构建
```

#### Q3: 无法安装到手机
```bash
# 解决方案：
1. 手机开启 USB 调试模式
2. 使用数据线连接电脑
3. 在 Android Studio 中选择设备运行
```

## 🎯 方案二：uni-app + HBuilder X（备选方案）

### ⚠️ 重要提示
此方案需要**大量代码重构**，仅在以下情况考虑：
- 必须使用 HBuilder X 的云打包
- 需要发布到小程序平台
- 团队熟悉 uni-app 生态

### 📋 迁移步骤

#### 1. 创建 uni-app 项目
在 HBuilder X 中：
- 文件 → 新建 → 项目
- uni-app → Vue3 → 默认模板
- 项目名称：Points

#### 2. 代码迁移清单
```
✅ src/pages/* → pages/*
✅ src/components/* → components/*
✅ src/stores/* → store/*
✅ src/utils/* → utils/*
✅ src/types/* → types/*
✅ src/App.vue → App.vue
✅ src/main.ts → main.js
❌ src/router/index.ts → 删除（改用 pages.json）
✅ src/assets/* → static/*
```

##### 3.2 API 适配
```javascript
// localStorage → uni.storage
localStorage.setItem('key', value) → uni.setStorageSync('key', value)
localStorage.getItem('key') → uni.getStorageSync('key')

// 路由跳转
router.push('/page') → uni.navigateTo({url: '/page'})
router.back() → uni.navigateBack()

// 提示框
alert('msg') → uni.showToast({title: 'msg'})
confirm('msg') → uni.showModal({content: 'msg'})
```

##### 3.3 Pinia 适配
uni-app 中使用 Pinia 需要特殊配置，参考 Pinia 官方文档。

#### 4. 使用 HBuilder X 打包

##### 云打包（推荐）
1. 菜单栏 → 发行 → 原生 app-云打包
2. 选择 Android 或 iOS
3. 等待 5-10 分钟
4. 下载 APK

##### 离线打包
1. 下载离线 SDK
2. 配置本地环境
3. 菜单栏 → 发行 → 原生 app-本地打包

### ⚠️ 注意事项
1. 部分 CSS3 特性不支持
2. 需要使用 uni-app 的 API 规范
3. 某些 npm 包可能不兼容
4. 需要全面测试所有功能
