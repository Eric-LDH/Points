# 🚀 Points 积分管理系统 - 5 分钟快速打包指南

## 📋 前提条件

### ✅ 已满足的条件
- Node.js 和 npm 已安装
- 项目依赖已安装完成
- Capacitor 已配置完成

### ⚠️ 需要自行准备
- **Android Studio**（推荐）或 Android SDK
- JDK 17（Android 构建需要）

---

## 🎯 三种打包方式（任选其一）

### 方式一：双击运行（最简单）⭐

**Windows 用户**：
```
直接双击：build-android.bat
```

这个脚本会自动：
1. ✅ 检查环境
2. ✅ 构建 Web 项目
3. ✅ 同步到 Android
4. ✅ 提示下一步操作

---

### 方式二：使用 npm 命令

```bash
# 完整构建流程（推荐）
npm run build:android

# 或者分步执行
npm run build          # 构建 Web 项目
npm run cap:android    # 同步到 Android
npm run cap:open:android  # 打开 Android Studio
```

然后在 Android Studio 中点击 Run 按钮即可。

---

### 方式三：手动命令

```bash
# 1. 构建
npm run build

# 2. 同步
npx cap sync android

# 3. 打开 Android Studio
npx cap open android
```

---

## 📱 获取 APK 文件

### 方法 A：Android Studio 构建
1. 在 Android Studio 中打开项目
2. 选择菜单：**Build > Build Bundle(s) / APK(s) > Build APK(s)**
3. 等待构建完成
4. APK 位置：`android\app\build\outputs\apk\debug\app-debug.apk`

### 方法 B：命令行构建（需 Android SDK）
```bash
cd android
.\gradlew assembleDebug
```

APK 会输出到相同位置。

## 🐛 遇到问题？

### ❌ "Unable to launch Android Studio"
**解决**: 未安装 Android Studio  
→ 下载安装：https://developer.android.com/studio

### ❌ Gradle 同步失败
**解决**: 
```bash
# 删除缓存重试
rm -rf android/.gradle
npx cap sync android
```

### ❌ 不知道下一步做什么
**解决**: 查看自动生成的提示文档  
→ 阅读：`README_MOBILE.md`
