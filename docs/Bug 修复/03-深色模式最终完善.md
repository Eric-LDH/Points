## ⚠️ 重要注意事项

### 1. 颜色对比度

**确保可读性**:
```scss
// ✅ 良好的对比度
.text-primary {
  color: #F9FAFB; // 在深色背景上清晰可读
}

.text-secondary {
  color: #D1D5DB; // 中等对比度
}

// ❌ 避免：对比度不足
.bad-contrast {
  color: #6B7280; // 在深色背景上难以阅读
}
```

### 2. !important 的使用

```scss
// ✅ 必要时使用
.header {
  background: var(--dark-card-bg) !important; // 必须覆盖原有背景
}

// ❌ 避免滥用
.card {

  background: var(--dark-card-bg);
}
```
