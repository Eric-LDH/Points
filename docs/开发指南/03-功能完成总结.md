**检查流程**:
```
用户点击兑换
    ↓
检查积分是否足够
    ↓
查询周期内已兑换次数
    ↓
比较是否超过限制
    ↓
未超限 → 允许兑换
已超限 → 显示提示（本周已兑 2/2 次）
    ↓
确认 → 扣除积分 → 创建记录
```

## 📝 使用说明

### 首次使用
1. 启动应用后会自动创建默认数据
2. 默认孩子：小明
3. 默认规则：晾衣服 (+2 分)、早读 20 分钟 (+3 分)

### 添加规则
1. 进入【我的】→【积分规则管理】
2. 点击右上角【✚ 新增】
3. 填写规则信息（图标自动生成）
4. 点击【✓ 保存】

### 添加兑换商品
1. 进入【我的】→【兑换商品管理】
2. 填写商品信息
3. 设置周期限制（每周/每月/每年）
4. 点击【✓ 保存】

### 备份数据
1. 进入【我的】页面
2. 滚动到【数据备份与恢复】
3. 点击【📤 手动备份】
4. 下载 JSON 文件

### 恢复数据
1. 进入【我的】页面
2. 点击【📥 从文件恢复】
3. 选择之前导出的 JSON 文件
4. 确认恢复
