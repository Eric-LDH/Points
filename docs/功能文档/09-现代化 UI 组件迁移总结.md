## 🎨 新旧对比

### Alert → Toast

#### ❌ 旧方式
```typescript
alert('保存成功')
```

#### ✅ 新方式
```typescript
showToast({ message: '保存成功', type: 'success' })
```

**优势：**
- ✓ 非阻塞式，用户体验更好
- ✓ 自动消失，无需手动关闭
- ✓ 美观的动画效果
- ✓ 支持多种类型（success/error/warning/info）
- ✓ 支持深色模式

---

### Confirm → ConfirmDialog

#### ❌ 旧方式
```typescript
if (confirm('确定删除？')) {
  deleteItem()
}
```

#### ✅ 新方式
```typescript
const confirmed = await showConfirm({
  title: '确认删除',
  message: '确定删除？此操作不可恢复',
  type: 'danger'
})
if (confirmed) {
  deleteItem()
}
```

**优势：**
- ✓ 异步 Promise，代码更清晰
- ✓ 自定义标题和消息
- ✓ 美观的模态对话框
- ✓ 支持不同类型（warning/danger/info）
- ✓ 更好的视觉层次

## 🎯 功能特性

### Toast 组件
- **四种类型**: success ✅, error ❌, warning ⚠️, info ℹ️
- **自动消失**: 默认 2 秒后自动关闭
- **动画效果**: 淡入淡出 + 位移动画
- **深色模式**: 完美支持深色模式
- **毛玻璃效果**: backdrop-filter 模糊背景

### ConfirmDialog 组件
- **三种类型**: warning ⚠️, danger ❗, info ℹ️
- **Promise 返回**: async/await 语法更简洁
- **自定义文本**: 可设置确定/取消按钮文字
- **大图标显示**: 醒目的警示图标
- **响应式设计**: 适配不同屏幕尺寸

## 📝 使用示例

### Toast 使用场景

```typescript
// 成功提示
showToast({ message: '添加成功！', type: 'success' })

// 错误提示
showToast({ message: '网络错误', type: 'error' })

// 警告提示
showToast({ message: '请先选择孩子', type: 'warning' })

// 信息提示
showToast({ message: '正在加载...', type: 'info' })

// 自定义时间
showToast({ 
  message: '操作成功', 
  type: 'success', 
  duration: 3000 
})
```

### Confirm 使用场景

```typescript
// 简单用法
const result = await showConfirm('确定要删除吗？')
if (result) {
  // 执行删除
}

// 完整配置
const result = await showConfirm({
  title: '删除确认',
  message: '确定删除？此操作不可恢复！',
  confirmText: '删除',
  cancelText: '取消',
  type: 'danger'
})
if (result) {
  // 执行删除
}
```

## 💡 最佳实践

### ✅ 推荐
```typescript
// 1. 操作成功后显示 Toast
await store.saveData()
showToast({ message: '保存成功！', type: 'success' })

// 2. 删除前使用 Confirm
const confirmed = await showConfirm({
  title: '删除确认',
  message: '确定删除？此操作不可恢复',
  type: 'danger'
})
if (confirmed) {
  await store.delete()
  showToast({ message: '删除成功', type: 'success' })
}

// 3. 表单验证失败用 Warning Toast
if (!formData.name) {
  showToast({ message: '请填写姓名', type: 'warning' })
  return
}
```

### ❌ 避免
```typescript
// ❌ 不要使用 alert/confirm
alert('保存成功')
if (confirm('确定？')) { }

// ❌ 不要在循环中使用 Toast
items.forEach(item => {
  showToast(`处理${item.name}`) // 会快速闪过多个提示
})
```
