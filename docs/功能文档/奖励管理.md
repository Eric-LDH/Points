## 🎯 使用流程

### 新增兑换商品
1. 进入【我的】→【兑换商品管理】
2. 点击右上角"✚ 新增"按钮
3. 填写表单：
   - 输入商品名称（如：手机时间 30 分钟）
   - 设置所需积分（如：50）
   - 选择周期限制类型
   - 设置周期内限制次数
   - 选择商品图标
   - （可选）填写商品描述
4. 点击"✓ 保存"
5. 商品添加到兑换列表

### 编辑兑换商品
1. 在商品列表中找到要编辑的商品
2. 点击"✎"编辑按钮
3. 修改表单内容
4. 点击"✓ 保存"
5. 更新立即生效

### 删除兑换商品
1. 在商品列表中找到要删除的商品
2. 点击"🗑"删除按钮
3. 确认删除对话框
4. 商品永久删除（不可恢复）

### 启用/禁用商品
1. 在商品列表中找到目标商品
2. 点击"✓/○"切换按钮
3. 绿色 ✓ 表示启用
4. 灰色 ○ 表示禁用
