# 现代化 UI 组件使用指南

## 📱 Toast 提示组件

### 功能说明
Toast 是一种非阻塞式的消息提示组件，用于向用户展示操作结果或状态信息。

### 使用方法

#### 1. 简单用法（默认 info 类型）
```typescript
import { showToast } from '@/utils/toast'

// 字符串形式
showToast('操作成功')

// 对象形式
showToast({
  message: '保存成功',
  type: 'success'
})
```

#### 2. 不同类型
```typescript
// 成功提示
showToast({
  message: '添加成功！',
  type: 'success'
})

// 错误提示
showToast({
  message: '网络错误，请重试',
  type: 'error'
})

// 警告提示
showToast({
  message: '请先选择孩子',
  type: 'warning'
})

// 信息提示
showToast({
  message: '正在加载...',
  type: 'info'
})
```

#### 3. 自定义显示时间（毫秒）
```typescript
showToast({
  message: '操作成功',
  type: 'success',
  duration: 3000 // 显示 3 秒
})
```

## ⚠️ Confirm 确认对话框

### 功能说明
Confirm 是模态对话框，用于需要用户确认的重要操作，会阻断后续操作直到用户做出选择。

### 使用方法

#### 1. 简单用法
```typescript
import { showConfirm } from '@/utils/confirm'

const result = await showConfirm('确定要删除吗？')
if (result) {
  // 用户点击确定
  deleteItem()
} else {
  // 用户点击取消
  console.log('已取消')
}
```

#### 2. 完整配置
```typescript
const result = await showConfirm({
  title: '删除确认',
  message: '确定要删除这个项目吗？\n此操作不可恢复！',
  confirmText: '删除',
  cancelText: '取消',
  type: 'danger' // 'warning' | 'danger' | 'info'
})

if (result) {
  // 执行删除操作
}
```

#### 3. 实际案例
```typescript
// 删除规则确认
const confirmed = await showConfirm({
  title: '确认删除',
  message: `确认删除"${rule.name}"？\n\n此操作不可恢复！`,
  type: 'danger'
})

if (confirmed) {
  store.deleteRule(rule.id)
  showToast({ message: '删除成功', type: 'success' })
}
```

## 📝 最佳实践

### ✅ 推荐做法
```typescript
// 1. 操作成功后显示 Toast
await store.saveData()
showToast({ message: '保存成功！', type: 'success' })

// 2. 删除前使用 Confirm
const confirmed = await showConfirm({
  title: '删除确认',
  message: '确定删除？此操作不可恢复',
  type: 'danger'
})
if (confirmed) {
  await store.delete()
  showToast({ message: '删除成功', type: 'success' })
}

// 3. 表单验证失败用 Warning Toast
if (!formData.name) {
  showToast({ message: '请填写姓名', type: 'warning' })
  return
}
```

### ❌ 避免做法
```typescript
// ❌ 不要使用 alert
alert('保存成功')

// ❌ 不要使用 confirm
if (confirm('确定删除？')) {
  // ...
}

// ❌ 不要在循环中使用 Toast
items.forEach(item => {
  showToast(`处理${item.name}`) // 会快速闪过多个提示
})
```

## 🔄 迁移指南

### 旧代码 → 新代码

#### Alert 迁移
```typescript
// 之前
alert('保存成功')

// 现在
showToast({ message: '保存成功', type: 'success' })
```

#### Confirm 迁移
```typescript
// 之前
if (confirm('确定删除？')) {
  deleteItem()
}

// 现在
const confirmed = await showConfirm('确定删除？')
if (confirmed) {
  deleteItem()
}
```
