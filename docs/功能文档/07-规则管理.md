## 🎯 使用流程

### 新增规则
1. 进入【我的】→【积分规则管理】
2. 点击右上角"✚ 新增"按钮
3. 填写表单：
   - 输入规则名称（如：晾衣服）
   - 选择分类（如：家务）
   - 选择类型（奖励/惩罚）
   - 输入积分值（如：2）
   - （可选）填写备注说明
4. 点击"✓ 保存"
5. 系统自动生成图标并保存到列表

### 编辑规则
1. 在规则列表中找到要编辑的规则
2. 点击"✎"编辑按钮
3. 修改表单内容
4. 点击"✓ 保存"
5. 更新立即生效

### 删除规则
1. 在规则列表中找到要删除的规则
2. 点击"🗑"删除按钮
3. 确认删除对话框
4. 规则永久删除（不可恢复）

## 📊 数据结构

### Rule 类型定义
```typescript
interface Rule {
  id: string
  name: string
  category: '家务' | '生活习惯' | '学习习惯' | '加分项' | '惩罚'
  type: 'reward' | 'penalty'
  points: number
  icon: string
  description?: string
  enabled: boolean
  sortOrder: number
  createdAt: string
  updatedAt: string
}
```
