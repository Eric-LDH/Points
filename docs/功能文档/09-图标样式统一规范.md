# 图标样式统一规范

## 📋 概述

将所有页面的列表项图标样式统一为兑换商品列表的 `item-row__icon` 标准样式，确保视觉一致性和专业性。

## ✅ 统一标准

### 标准图标样式（item-row__icon）

```scss
&__icon {
  font-size: 32px;           // 图标字体大小
  width: 40px;               // 固定宽度
  height: 40px;              // 固定高度
  display: flex;             // Flexbox 布局
  align-items: center;       // 垂直居中
  justify-content: center;   // 水平居中
  flex-shrink: 0;            // 防止压缩
}
```

### 样式特点

1. **固定尺寸容器** - `40px × 40px` 确保所有图标占据相同空间
2. **Flexbox 居中** - 图标在容器中完美居中
3. **防止压缩** - `flex-shrink: 0` 保证图标不会被挤压变形
4. **适中大小** - `32px` 字体大小既清晰又不突兀

## 🎨 修改前后对比

### 修改前（不统一）

```scss
// record.vue - rule-item__icon
&__icon {
  font-size: 28px;  // ❌ 尺寸不一
}

// index.vue - rule-item__icon
&__icon {
  font-size: 32px;  // ❌ 无固定容器
}

// stats.vue - category-item__icon
&__icon {
  font-size: 24px;  // ❌ 尺寸偏小
}

// stats.vue - record-item__icon
&__icon {
  font-size: 28px;  // ❌ 尺寸不一
}
```

### 修改后（统一标准）

```scss
// 所有页面的图标样式完全一致
&__icon {
  font-size: 32px;        // ✅ 统一大小
  width: 40px;            // ✅ 固定宽度
  height: 40px;           // ✅ 固定高度
  display: flex;          // ✅ Flexbox 布局
  align-items: center;    // ✅ 垂直居中
  justify-content: center;// ✅ 水平居中
  flex-shrink: 0;         // ✅ 防止压缩
}
```

## 🎯 视觉效果提升

### 统一前的问题
1. **尺寸不一致** - 不同页面图标大小不一（24px/28px/32px）
2. **容器不固定** - 图标会随内容压缩变形
3. **对不齐** - 没有固定宽高导致布局偏移
4. **不够专业** - 缺乏统一的视觉规范

### 统一后的优势
1. **视觉一致性** - 所有图标大小、间距完全统一
2. **布局稳定** - 固定尺寸确保对齐精准
3. **抗压缩** - `flex-shrink: 0` 防止变形
4. **专业美观** - 统一的视觉语言提升品质感
