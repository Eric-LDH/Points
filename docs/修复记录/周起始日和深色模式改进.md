## 问题 1：本周表现统计周期优化

### 🔍 问题分析

**修改前 ❌**:
- 使用"滚动最近 7 天"的方式计算
- 从今天往前推 6 天，导致每周的起始日不固定
- 不符合用户习惯（通常认为一周从周一开始）

**示例问题**:
```
假设今天是周三（2026-03-13）
❌ 修改前：显示 3/7(周五) → 3/13(周三)
✅ 修改后：显示 3/9(周一) → 3/13(周三)
```

### ✅ 完整解决方案

**修改文件**: [src/pages/stats/stats.vue](file://d:\Project\Points\src\pages\stats\stats.vue#L135-L168)

**修改前**:

**修改后**:
```typescript
// 计算本周数据（周一到周日）- 基于当前孩子的记录
const weekDays = computed(() => {
  const childId = store.currentChildId
  const childRecords = pointsRecords.value.filter(r => r.childId === childId)
  
  const today = new Date()
  const currentDay = today.getDay() // 0(周日) - 6(周六)
  
  // 计算本周一的日期
  const mondayOffset = currentDay === 0 ? 6 : currentDay - 1 // 如果是周日，往回推 6 天；否则往回推 (currentDay-1) 天
  const monday = new Date(today)
  monday.setDate(monday.getDate() - mondayOffset)
  monday.setHours(0, 0, 0, 0)
  
  const result = []
  const weekDayNames = ['周一', '周二', '周三', '周四', '周五', '周六', '周日']
  
  // 从周一开始，获取 7 天的数据
  for (let i = 0; i < 7; i++) {
    const date = new Date(monday)
    date.setDate(date.getDate() + i)
    const dateStr = date.toISOString().split('T')[0]
    
    // 如果这一天还没到（未来），跳过
    if (date > today) {
      break
    }
    
    // 计算这一天的总积分
    const dayRecords = childRecords.filter(r => r.date === dateStr)
    const dayPoints = dayRecords.reduce((sum, r) => sum + r.points, 0)
    
    result.push({
      name: weekDayNames[i],
      points: dayPoints,
      date: dateStr
    })
  }
})
```

### 📊 核心算法说明

#### 1. 计算本周一的日期

```typescript
const currentDay = today.getDay() // 获取今天是周几（0-6，0 是周日）

// 计算需要往回推几天才能到周一
const mondayOffset = currentDay === 0 ? 6 : currentDay - 1
// 解释：
// - 周一 (1): offset = 0，不需要回推
// - 周二 (2): offset = 1，回推 1 天
// - 周三 (3): offset = 2，回推 2 天
// - ...
// - 周日 (0): offset = 6，回推 6 天

const monday = new Date(today)
monday.setDate(monday.getDate() - mondayOffset)
monday.setHours(0, 0, 0, 0) // 设置为当天的 00:00:00
```

#### 2. 从周一开始遍历 7 天

```typescript
for (let i = 0; i < 7; i++) {
  const date = new Date(monday)
  date.setDate(date.getDate() + i) // 从周一开始，依次 +0, +1, +2...+6
  
  // 如果这一天还没到（未来），跳过
  if (date > today) {
    break
  }
  
  // ... 计算这一天的积分
}
```

## 💡 核心技术要点

### 1. 自然周计算算法

```typescript
// 关键步骤：
// 1. 获取今天是周几
const currentDay = today.getDay() // 0(周日) - 6(周六)

// 2. 计算本周一的日期
const mondayOffset = currentDay === 0 ? 6 : currentDay - 1
const monday = new Date(today)
monday.setDate(monday.getDate() - mondayOffset)

// 3. 从周一开始遍历 7 天
for (let i = 0; i < 7; i++) {
  const date = new Date(monday)
  date.setDate(date.getDate() + i)
  
  // 未来的日期不显示
  if (date > today) {
    break
  }
}
```
