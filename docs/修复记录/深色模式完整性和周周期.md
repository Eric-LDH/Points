## 问题 1：本周表现显示完整一周

### 🔍 问题分析

**修改前 ❌**: 
```typescript
// 如果这一天还没到（未来），跳过
if (date > today) {
  break // 直接跳出循环
}
```

**问题**：
- 今天是周三，只显示周一、周二、周三
- 周四到周日不显示
- 柱状图不完整

**修改后 ✅**:
```typescript
// 如果这一天还没到（未来），显示 0 分
let dayPoints = 0
if (date <= today) {
  // 计算这一天的总积分
  const dayRecords = childRecords.filter(r => r.date === dateStr)
  dayPoints = dayRecords.reduce((sum, r) => sum + r.points, 0)
}

result.push({
  name: weekDayNames[i],
  points: dayPoints, // 未来日期为 0
  date: dateStr
})
```

### 📊 效果对比

| 场景 | 修改前 ❌ | 修改后 ✅ |
|------|----------|----------|
| 今天是周三 | 显示：周一、周二、周三<br>柱子数量：3 根 | 显示：周一、周二、周三、周四、周五、周六、周日<br>柱子数量：7 根（周四到周日为 0 分） |
| 今天是周日 | 显示：周一到周日<br>柱子数量：7 根 | 显示：周一到周日<br>柱子数量：7 根（完整） |
| 今天是周一 | 显示：周一<br>柱子数量：1 根 | 显示：周一、周二、周三、周四、周五、周六、周日<br>柱子数量：7 根（周二到周日为 0 分） |

## ⚠️ 重要注意事项

### 1. 性能考虑

**避免过度嵌套**:
```scss
// ✅ 好的做法：控制在 4 层以内
html.dark {
  .page {
    .card {
      .content {
        .text {
          // OK
        }
      }
    }
  }
}

// ❌ 避免：嵌套过深，难以维护
html.dark {
  .page {
    .card {
      .content {
        .section {
          .paragraph {
            .text {
              .span {
                // 太深了！
              }
            }
          }
        }
      }
    }
  }
}
```

### 2. 颜色对比度

**确保可读性**:
```scss
// ✅ 良好的对比度
.text-primary {
  color: #F9FAFB; // 在深色背景上清晰可读
}

.text-secondary {
  color: #D1D5DB; // 中等对比度
}

// ❌ 避免：对比度不足
.bad-contrast {
  color: #6B7280; // 在深色背景上难以阅读
}
```

### 3. !important 的使用

```scss
// ✅ 必要时使用
.header {

.card {

  background: var(--dark-card-bg);
}
```
