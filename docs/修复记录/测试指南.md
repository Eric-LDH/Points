## 📋 安装步骤

### 方法 1：通过 USB 数据线安装

1. **启用开发者模式**：
   - 打开手机设置 → 关于手机
   - 连续点击"版本号"7 次
   - 返回设置 → 系统 → 开发者选项
   - 开启"USB 调试"

2. **连接电脑**：
   - 使用 USB 数据线连接手机和电脑
   - 手机上允许 USB 调试授权

3. **安装 APK**：
   ```bash
   # 在项目根目录执行
   adb install android/app/build/outputs/apk/debug/app-debug.apk
   ```

4. **验证安装**：
   ```bash
   adb shell pm list packages | grep points
   # 应显示：package:com.points.family
   ```

### 方法 2：直接传输安装

1. **复制 APK 到手机**：
   - 将 `app-debug.apk` 文件复制到手机存储

2. **在手机上安装**：
   - 使用文件管理器找到 APK 文件
   - 点击安装
   - 如果提示"未知来源"，需要允许安装未知应用

## 🐛 可能的问题及解决方案

### 问题 1：备份时没有弹出分享对话框

**可能原因**：
- Share 插件未正确集成
- Android 版本太低

**解决方案**：
```bash
# 重新同步插件
npx cap sync android

# 检查插件是否安装
npm list @capacitor/share
```

**临时方案**：
如果 Share 插件确实无法使用，可以添加存储权限使用浏览器下载方式：

编辑 `android/app/src/main/AndroidManifest.xml`，添加：
```xml
<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE" />
<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE" />
```

然后重新构建：
```bash
npm run build:android
```

---

### 问题 2：深色模式重启后未恢复

**检查步骤**：
1. 打开浏览器开发者工具（如果是 Web 版本）
2. 查看 localStorage 中是否有 `points_darkMode` 键
3. 值应该是 `"true"` 或 `"false"`

**调试方法**：
在控制台输入：
```javascript
localStorage.getItem('points_darkMode')
```

**预期输出**：
- `"true"` - 深色模式已开启
- `"false"` - 浅色模式已开启
- `null` - 从未切换过（默认浅色）

---

### 问题 3：状态栏仍然显示不同颜色

**可能原因**：
- 手机系统定制 ROM 限制了状态栏样式
- 某些 MIUI、EMUI 等需要特殊处理

**解决方案**：
编辑 `android/app/src/main/res/values/styles.xml`，确保：
```xml
<item name="android:statusBarColor">@android:color/transparent</item>
<item name="android:windowLayoutInDisplayCutoutMode">shortEdges</item>
```

并检查 `MainActivity.java` 是否需要添加额外代码。
