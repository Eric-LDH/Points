**修改文件**：
- `android/app/src/main/res/values/colors.xml` (新建)
- `android/app/src/main/java/com/points/family/MainActivity.java`

**效果**：

### 3. 手动备份 JSON 转换错误

**问题描述**：在手机上点击手动备份时报错：`org.json.JSONObject cannot be cast to java.lang.String`

**问题根源**：Share 插件的参数传递方式不正确，对象结构导致类型转换失败。

**解决方案**：
- ✅ 重构 `exportToFile()` 方法
- ✅ 直接在 share() 调用中传入参数对象，避免中间变量
- ✅ 优化代码结构，减少嵌套层级

**修改文件**：
- `src/utils/storage.ts`

**修改前**：
```typescript
const shareData = {
  title: '备份积分数据',
  text: '这是您的积分管理数据备份',
  files: [{...}]
}
await Capacitor.Plugins.Share.share(shareData)
```

**修改后**：
```typescript
await Capacitor.Plugins.Share.share({
  title: '备份积分数据',
  text: '这是您的积分管理数据备份',
  files: [{...}]
})
```

**效果**：
- 备份功能正常工作
- 弹出系统分享对话框
- 可以保存到文件管理器或分享到其他应用
